Curso de Programação Web

Em 2018 fora ofertado pelo Programa Nacional de Acesso ao Ensino Técnico e Emprego (PRONATEC), no Instituto Federal do Paraná - Campus Paranavaí o curso de Programação web (duração de 6 meses) tanto para a comunidade interna quanto para a comunidade externa. Este curso tinha como objetivo ensinar os princípios básicos do desenvolvimento web. Para tanto, fora ensinado o básico de HTML, CSS, JavaScript e PHP para a criação de um site simples. Além disso, esse curso serviu para a formação profissional de muitas pessoas da área de T.I.

Curso de Administrador de Banco de Dados

Curso online de Administrador de Banco de Dados ofertado pelo Instituto Federal de Rondônia - Campus Ji-Paraná (2020). Este curso teve como objetivo apresentar os princípios de como administrar um banco de dados. Além disso, fora ensinado as três formas de modelagem, ou seja, o Modelo Conceitual, o Diagrama Entidade e Relacionamento (DER) e o Modelo Lógico. A ferramenta utilizada para a diagramação do DER e para a criação do script SQL fora o WorkBench.

Curso de Gestão de Microempresa

Curso FIC de Gestão de Microempresa ofertado pelo Instituto Federal do Rio Grande do Sul - Campus Sul-Rio-Grandense (2020), cujo objetivo era descrever qual a melhor forma de gerir micro empresas, assim como entender a economia(oferta e demanda) e de modo a identificar como as mudanças na economia podem afetar uma micro empresa.

Projeto - Sistema para Gestão de Associações

Participação em um projeto de desenvolvimento de software em grupo no Instituto Federal do Paraná - Campus Paranavaí, sob a coordenaçaõ do Prof. Frank William e do Prof. Marcelo Terenciani, cujo objetivo é desenvolver um sistema para auxiliar a gestão de administrativas de associações, tendo como estudo de caso a AMAER (Associação Maringaense de Aeromodelismo), utilizando Node JS, React e MongoDB.

A Tecnologia da Informação e Comunicação como Recurso Didático para o Ensino da Multiplicação: DominóMath

O presente artigo cujo tema é: "A Tecnologia da Informação e Comunicação como Recurso Didático para o Ensino da Multiplicação: DominóMath", publicado nos periódicos de 2019 da Revista Mundi no qual possui classificação B4, tem como objetivo apresentar o protótipo de um jogo, Dominó Matemático, que tem por finalidade servir como uma ferramenta didática e pedagógica capaz de contribuir para o processo de ensino e aprendizagem da operação de multiplicação de alunos que estudam no Ensino Fundamental, uma vez que, embora esta operação seja decorrente das repetidas operações de adições de parcelas iguais, muitos alunos ainda possuem dificuldades em entender, compreender e aplicar a multiplicação.
